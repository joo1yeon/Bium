# 📌기능 소개

## 주요 기능

1. 실시간 라이브 스트리밍 비움

2. 회원 관리 (게임 기록 관리, 회원 정보 관리, 랭크 조회)

3. face-api.js 연동 - 카메라 이탈 인지, 표정 인지

## 세부 기능

- 메인
  - 게임방 목록, 생성
  - 배경 영상과 개인 웹캠 영상 송출
  - 타이머
- 회원
  - 자체 로그인
  - 비밀번호 찾기
  - 비움 기록 확인
  - 마이페이지
